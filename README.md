# Calculadora Angular

## 📌 Descripción del proyecto
Calculadora que sume dos números enteros.
Hay diferentes botones en la calculadora:
- Botón números: 0 al 9.
- Botón "+": debe aparecer en pantalla.
- Botón "C": al pulsarlo debe limpiar la pantalla.
- Botón "=": debe devolver la suma de los dos números.

## Ejecutar el proyecto
Introducir la siguiente línea de comando: 

```bash
ng serve
```
